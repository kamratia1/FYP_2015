INSTALLATION - README:
========================

1. Please do not connect the PicoProg programmer to the PC before asked by the software

2. Run setup.exe from the PCap02plus_Evaluation_and_Assembler

3. Step 2 usually installs the driver for the PicoProg. Only if this fails, please perform setup.exe from the Driver folder.
